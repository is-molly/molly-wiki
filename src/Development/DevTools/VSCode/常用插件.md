# 常用插件

## 插件列表

```shell
# vs Markdown工具
Markdown All in One

# 实时预览markdown
Markdown Preview Enhanced

# 文件比较（选中一代码右键Select Text for Compare，选中另外一部分代码右键Compare Text with Previous Selection即可）
Partial Diff

# TODO tab
TODO Tree

# 图标插件
vscode-icons

# 括号显示不同颜色
Bracket Pair Colorizer

# 注释信息人性化（根据告警、查询、TODO、高亮等标记对注释进行不同的展示。此外，还可以对注释掉的代码进行样式设置）
Better Comments

# 用于代码的上下对齐（它能够用冒号（：）、赋值（=，+=，-=，*=，/=）和箭头（=>）对齐代码）
# 使用方法：Ctrl+Shift+p输入“Align”确认即可。
Better Align

# 路径图像预览
Image preview

# 显示当前文件大小
file-size 

# 代码运行环境(*)
code Runner

# eclipse快捷键
Eclipse Keymap

# 智能代码提示插件(和别的插件不同的是，采用的是 AI 技术，吸收了 github 上成千上万的优秀代码，还会结合你日常的编码习惯不断学习进化)
Path Intellisense

# git 功能增强插件，鼠标放到代码行上，每一行代码的变动都一清二楚~
GitLens

# 查看 git 和文件历史记录
Git History 

# 一款好看 vscode 的主题
One Dark Pro
```
